\subsection{ADT}
\label{sec:adt}
	O ambiente de desenvolvimento do Android SDK, disponibiliza as ferramentas
necessárias para começar a desenvolver as aplicações na plataforma utilizando a linguagem de 
programação Java. 

	O SDK inclui funcionalidades utéis na criação de aplicativos como por exemplo emuladores 
de aparelhos, ferramentas de depuração, visualização da utilização de memória, análise de desempenho, 
dentre outras. 

	O ADT \footnote{ADT: \url{http://developer.android.com/tools/sdk/eclipse-adt.html}} é 
um plugin para a IDE Eclipse\footnote{Eclipse: \url{https://www.eclipse.org/}} disponibilizado pela Google para o desenvolvimento de aplicativos android.
