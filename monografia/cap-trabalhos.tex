\chapter{Trabalhos Futuros}
\label{cap:trabalhos_futuros}

Este projeto pode ser melhorado e ampliado através de novas funcionalidades e melhorias. Algumas delas são:

\begin{itemize}
\item{Imagens:} De maneira geral as imagens do jogo podem ser melhoradas.
\item{Sons:} Não foram implementados sons no jogo. Uma implementação seria uma grande melhora.
\item{Especiais:} Implementar possíveis especiais a serem gastos durante o jogo. Algo como mostrar a primeira
letra do nome da personalidade por exemplo.
\item{Tela de compra:} Implementar a economia do jogo, vendendo moedas que serviriam para a compra de especiais.
\item{Tela de Estatísticas:} Implementar a tela no jogo que mostra as estatísticas persistidas na tabela do banco de dados Historico\_Estatistica
\end{itemize}
